# 🛒 Orderly – Marketplace Pre-Order Mahasiswa

## 📌 Deskripsi Project

**Orderly** adalah aplikasi web berbasis marketplace yang dirancang khusus untuk mempermudah mahasiswa dalam melakukan transaksi jual beli dengan sistem **Pre-Order (PO)** secara terstruktur.

Aplikasi ini hadir sebagai solusi dari permasalahan transaksi manual di grup chat yang sering menimbulkan kesalahan pencatatan, keterlambatan informasi, serta kurangnya transparansi antara penjual dan pembeli.

Dengan Orderly, seluruh proses mulai dari pemesanan, pembayaran, hingga pemantauan status pesanan dapat dilakukan dalam satu platform yang rapi, mudah digunakan, dan transparan.

---

## 🎯 Tujuan Aplikasi

* Mempermudah sistem pre-order di lingkungan mahasiswa
* Mengurangi kesalahan transaksi manual
* Meningkatkan transparansi antara buyer dan seller
* Menyediakan platform jual beli yang lebih terorganisir

---

## 👥 Role Pengguna

1. **Admin**

   * Mengelola user
   * Verifikasi akun seller

2. **Seller (Penjual)**

   * Mengelola produk
   * Menentukan sistem pre-order (deadline & stok)
   * Verifikasi pembayaran

3. **Buyer (Pembeli)**

   * Melihat produk
   * Melakukan pemesanan
   * Upload bukti pembayaran
   * Memberikan rating & review

---

## ⚙️ Fitur Utama

* Landing page (tanpa login)
* Sistem autentikasi (Register & Login)
* Manajemen produk (CRUD)
* Sistem Pre-Order (deadline & stok)
* Pemesanan produk
* Upload bukti pembayaran
* Status pesanan (Pending, Dibayar, Diproses, Selesai)
* Sistem rating & review

---

## 🛠️ Teknologi yang Digunakan

* **Frontend**: HTML, CSS, Bootstrap
* **Backend**: Node.js (Express.js)
* **Database**: MySQL
* **Authentication**: JWT
* **File Upload**: Multer

## 🚀 Cara Menjalankan Project

### 1. Clone Repository

```
git clone https://github.com/username/orderly.git
cd orderly
```

### 2. Install Dependencies (Backend)

```
cd backend
npm install
```

### 3. Jalankan Server

```
npm run dev
```

### 4. Akses Frontend

Buka browser:

```
http://orderly.web.id
```
